# Milvus Lite REST Wrapper 项目说明

## 项目概述
**milvus-lite-rest-wrapper** 是一个为 Milvus Lite 提供 RESTful API 封装的项目，旨在提供一个轻量化的本地向量数据库解决方案。

## 平台支持
- **支持**: macOS、Linux
- **不支持**: Windows

## 项目特点
- **轻量化**: 基于 milvus-lite 实现本地化运行
- **RESTful API**: 提供标准的 REST 接口访问 Milvus 功能
- **本地存储**: 支持本地数据库文件存储

## 依赖说明
项目依赖于 milvus-lite 项目，这是一个轻量化的 Milvus 版本，专为本地和边缘部署设计。

## 用途
- 适合需要向量搜索功能但不想部署完整 Milvus 集群的场景
- 本地开发和测试环境
- 轻量级嵌入式向量数据库应用

## 本地化运行优势
- 无需复杂的服务器部署
- 低资源占用
- 快速启动和响应
- 适合单机应用场景

这个项目为需要向量数据库功能但追求轻量化和本地化部署的开发者提供了一个很好的解决方案。
